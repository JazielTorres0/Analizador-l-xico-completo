Este proyecto implementa un analizador léxico en Python que reconoce los símbolos léxicos definidos en el archivo simbolos_lexicos.pdf, sin utilizar expresiones regulares (re).
El analizador procesa una cadena de código fuente y genera una lista de tokens, cada uno acompañado de su tipo numérico, según la especificación del lenguaje.
